# Community

Konum tabanlı sosyal mobil uygulama — mikroservis mimarisi.

> **Asıl proje `community/` dizinindedir.** Güncel dokümantasyon, mimari,
> kurulum ve API bilgisi için: [community/README.md](community/README.md)
>
> Bu dosya bilerek kısa tutulur; içerik burada tekrarlanırsa güncelliğini
> yitirir. Mimari diyagramlar `architecture/` klasöründedir.

```bash
cd community
cp .env.example .env   # değerleri doldur (bkz. community/README.md)
make up
```
